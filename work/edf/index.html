As an exploration of visual storytelling, I decided to design this concept marketing webpage for Earth Defense Force 5, a sci-fi action game released in 2017. I focused on marketing the game's unique features and fast-paced action, using official game screenshots to support the energetic visual design.

My process started with establishing the project objective and deciding on content flow by creating an information architecture diagram and a quick wireframe in Figma. After the content structure had been roughly laid-out, I moved on to establishing visual design.

To convey emotions and aesthetics that represented the Earth Defense Force brand, I created a moodboard that included the game assets alongside media with similar aesthetics. I used the moodboard as a guide for font choice, visual elements / UI, and copy direction as I mocked up the web page. I derived color from the game assets and used game assets themselves to support the copy.
