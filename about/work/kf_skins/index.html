Here is a collection of custom weapon skins I've done for the horde survival shooter game, Killing Floor 2, through the Steam Workshop.

The process involved installing the Killing Floor 2 Software Development Kit and using it to download necessary weapon assets and textures/skins. I edited the skin files in Photoshop and re-applied it onto the weapon models with Unreal Editor built into the Killing Floor 2 SDK. I then set up a dedicated test server to live test the newly skinned weapons in-game.
